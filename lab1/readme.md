## Skrypt odpalający serwer i clientów

```
python3 main.py
```

## Klient

- `>` domyśnie wiadomość przez TCP
- `u >` wiadomość przez UDP
- `u > ascii >` wysłanie zawartości pliku ascii_art.txt przez UDP
- `m >` wiadomość przez multicast
- `c` wyjście z programu
